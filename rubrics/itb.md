## Lab requirements

The following requirements describe the necessary deliverables for the Security Research Lab.

| Criteria                                                                                                              | Points |
| --------------------------------------------------------------------------------------------------------------------- | ------ |
| The report includes an overview of the attack method and the incident you investigated.                               | 8      |
| The report includes details about the importance and relevance of the attack method and incident.                     | 8      |
| The report includes technical information about how the attack.                                                       | 8      |
| The report includes the attack's objective.                                                                           | 8      |
| The report includes risks posed to the victim's organization by the attack.                                           | 8      |
| The report includes a timeline of the attack.                                                                         | 8      |
| The report includes details about the tactics and techniques used in the attack.                                      | 8      |
| The report includes information about the impact of the attack.                                                       | 8      |
| The report includes details about the aftermath of the attack.                                                        | 8      |
| The report includes the lessons that the cybersecurity field can learn from this attack.                              | 8      |
| The report includes best practices that could have been implemented to prevent this attack.                           | 8      |
| The report is formatted in a clear manner.                                                                            | 5      |
| The report was shared publicly so that it is able to be viewed by the instructor on their first attempt when grading. | 2      |
| The report uses clear and professional language.                                                                      | 5      |
| **Total possible points**                                                                                             | 100    |
