## Lab requirements

The following requirements describe the necessary deliverables for the Security Research Lab.

| Criteria                                                                                          | Points |
| ------------------------------------------------------------------------------------------------- | ------ |
| Your name and the presentation's title are present on an introductory slide in your presentation. | 5      |
| The slide deck's content supports the verbal presentation instead of being a script.              | 10     |
| Your presentation is delivered to the class at the assigned date and time.                        | 10     |
| The presentation is delivered within the time limit provided.                                     | 5      |
| Visual elements and design in the presentation is professional.                                   | 5      |
| The report includes an overview of the attack method and the incident you investigated.           | 5      |
| The report includes details about the importance and relevance of the attack method and incident. | 5      |
| The report includes technical information about how the attack.                                   | 5      |
| The report includes the attack's objective.                                                       | 5      |
| The report includes risks posed to the victim's organization by the attack.                       | 5      |
| The report includes a timeline of the attack.                                                     | 5      |
| The report includes details about the tactics and techniques used in the attack.                  | 5      |
| The report includes information about the impact of the attack.                                   | 5      |
| The report includes details about the aftermath of the attack.                                    | 5      |
| The report includes the lessons that the cybersecurity field can learn from this attack.          | 5      |
| The report includes best practices that could have been implemented to prevent this attack.       | 5      |
| The report is formatted in a clear manner.                                                        | 5      |
| The report uses clear and professional language.                                                  | 5      |
| **Total possible points**                                                                         | 100    |
